# Task: Project Setup & Scaffolding

## Description
Initialize the Vite project with vanilla JS, create the full directory structure, set up the HTML shell with the dark theme layout skeleton, and define all CSS custom properties for the nightclub aesthetic.

## Acceptance Criteria
- [ ] Vite project initialized with vanilla JS template
- [ ] Directory structure matches epic spec (src/audio, src/midi, src/visuals, src/lessons, src/ui, src/styles, public/assets)
- [ ] index.html has the full layout skeleton (top bar, dual waveform area, mixer area, lesson panel)
- [ ] CSS custom properties defined: --bg-primary (#0a0a0a), --bg-secondary (#1a1a1a), --bg-tertiary (#2a2a2a), --accent-a (#00f0ff), --accent-b (#ff00e5), --text-primary, --text-secondary
- [ ] Glow utility classes for box-shadow effects with accent colors
- [ ] Vite dev server runs without errors
- [ ] Base font loaded (monospace/system for now)

## Technical Details
- Use `npm create vite@latest . -- --template vanilla`
- vite.config.js with no special plugins needed
- CSS organized into main.css (variables + resets), waveform.css, mixer.css, lessons.css
- HTML uses semantic sections: header.top-bar, section.waveforms, section.mixer, section.lesson-panel
