# Task: App Shell & Theming

## Description
Build the app shell layout, top bar with logo and MIDI connection status, lesson navigation, and apply the full dark nightclub theme. Create the overall page layout that positions waveforms, mixer, and lesson panel. Add the MIDI connection indicator with pulsing animation.

## Acceptance Criteria
- [ ] Top bar with "Onset" logo/wordmark (styled text, no image needed)
- [ ] MIDI connection indicator: pulsing green dot when connected, red when disconnected, gray when no permission
- [ ] Lesson navigation: tabs or buttons for "Lessons" and "Free Play" modes
- [ ] Main layout: flexbox/grid with waveforms top, mixer middle, lesson panel bottom
- [ ] Lesson panel is collapsible (hidden in free play mode)
- [ ] Full dark theme applied: #0a0a0a background, #1a1a1a panels, #2a2a2a borders
- [ ] Neon glow effects on interactive elements (buttons glow on hover)
- [ ] Smooth transitions on all state changes (panel collapse, mode switch)
- [ ] Minimum width: 1280px, no mobile layout (show "use desktop" message below 1280px)
- [ ] File drop zone: visual feedback when dragging audio files over deck areas

## Technical Details
- Layout: CSS Grid for the main 3-row layout (top-bar, content, lesson-panel)
- Content area: CSS Grid with 2 columns (Deck A | Deck B) for waveforms, single row mixer below
- Top bar: fixed position, z-index above content
- Connection indicator: CSS animation (pulsing box-shadow on a circle element)
- Collapsible panel: CSS transition on max-height with overflow hidden
- File drop: drag events on deck containers with visual hover state
