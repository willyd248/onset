Frequently asked questions

Which controllers are supported?

expand_more

Any controller with WebMIDI support works — most modern gear made after 2015. Tested: Hercules Inpulse 200/300/500, Pioneer DDJ-200/400/FLX4, Numark Party Mix, Denon MC3000. If your controller shows up in Chrome's Web MIDI settings, it works in onset. No drivers or extra software needed.

Is it free?

expand_more

Yes — core features are free forever. No credit card, no trial period, no paywall on fundamentals. The 30-lesson curriculum, real-time scoring, XP system, and spaced repetition all stay free. We'll offer advanced lesson packs and deeper analytics later.

Do I need to install anything?

expand_more

Nothing. onset runs entirely in your browser using the Web Audio API and WebMIDI API — no plugins, no extensions, no downloads. Open Chrome or Edge on a desktop (1200px+ screen), plug in your controller, and start. Safari has limited MIDI support so Chrome is recommended.

Do I need a controller to get started?

expand_more

No — mouse and keyboard work for basic lessons. You can learn tempo matching, phrasing theory, and EQ concepts without hardware. But a ~$100 controller makes everything 10x more real. The scoring system is designed around actual MIDI input. We're also building a phone-as-controller feature.

What music can I practice with?

expand_more

Any audio files you own — MP3, WAV, OGG, FLAC. Drag them onto a deck and go. onset auto-detects BPM and generates a beat grid. We don't host music for legal reasons, but any files you own work instantly.

What genres does onset teach?

expand_more

The core curriculum is genre-agnostic — beatmatching, EQ blending, transitions, and phrasing work the same whether you're mixing house, hip-hop, or drum & bass. Advanced lessons cover genre-specific techniques: rolling drops for techno, choppy cuts for hip-hop, long blends for house.
