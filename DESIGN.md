# Onset DJ App — Design System

## Creative North Star: "The Sonic Playground"

A DJ learning app that feels like a premium physical toy, not intimidating pro software.
Duolingo-inspired: safe for failure, ripe for play. White backgrounds, soft rounded cards,
playful color accents. Music learning app aesthetic.

## Spacing

Generous — "over-spacing" for premium feel:
- Card padding: 2rem–2.75rem
- Internal gaps: 1rem
- Section gaps: 1.5rem–2rem

## Component Rules

### No-Line Rule
Never use 1px borders to section content. Define boundaries with surface color transitions only.

### Controls
- **Knobs:** Circular, `surface-container-high` background, primary dot indicator
- **Sliders:** `full` rounded tracks, large grabbable handles
- **Crossfader:** Full-width horizontal, white handle with primary border

### Cards
No dividers. Separate items with surface color alternation or spacing.

### Buttons
- Primary: `--radius-xl`, primary background, on-primary text, no border
- Outline: White bg, 2px border, surface-container-high border color
- Transport: `--radius-xl`, 48px square icon buttons
